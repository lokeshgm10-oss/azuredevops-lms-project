# Enterprise LMS DevOps Project

This project demonstrates an end-to-end DevOps implementation for a Learning Management System (LMS) using Azure DevOps and cloud-native technologies.

## Technologies

- Azure DevOps
- GitHub
- Docker
- Kubernetes (AKS)
- Terraform
- SonarQube
- Trivy
- Azure Container Registry
- Azure Monitor

## CI/CD Workflow

```text
GitHub
   ↓
Azure DevOps
   ↓
SonarQube
   ↓
Trivy
   ↓
Docker
   ↓
ACR
   ↓
Terraform
   ↓
AKS
```
